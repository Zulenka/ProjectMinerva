Minerva Additional Info

Minerva is a Torn userscript for activity tracking, manual target lists, alerts, and a floating tracker widget.

Use the sections below to expand the information you need.

Quick Start
  1. Open a Torn profile page.
  2. Enter your API key when Minerva asks.
  3. Click Track Current.
  4. Set your preferred inactive threshold.
  5. Use the corner widget to monitor tracked targets.
Main Panel

Minerva shows a profile-page panel with status, next-ping countdown, and controls.

On your own profile page, settings can be opened from the gear button to keep the panel compact.

  • Track Current / Untrack Current: Add or remove the current profile from your tracked list
  • Reset API Key: Replace your stored Torn API key
  • View Toast: Preview the in-page Minerva toast
  • Clear Tracked: Clear the tracked list (keeps current target)
  • Show Logs: Open or close the Minerva log panel
Statuses and Timing

Minerva uses Torn activity data and your threshold setting to classify targets.

  • ACTIVE: Online or recently active (threshold-based for the main status)
  • INACTIVE <Xm: Not online, but still within your threshold window
  • INACTIVE Xm+: Not online and beyond your threshold window
  • NO TARGETS, NO API KEY, ACTIVITY UNAVAILABLE: Tracking/setup issues

When Torn reports a target as online, Minerva now displays Online now in the row instead of stale last-action text.

Corner Widget (Floating Tracker)

The corner widget stays available across Torn pages and shows your tracked list.

  • Overall Minerva status pill
  • Threshold display
  • Tracked rows (name, status, last activity text)
  • Manual ping cooldown badge when rate-limited

Widget controls include lock/unlock, compact/expand, and hide/reopen. Position is saved.

Tracked Row Actions
  • Username: Click to open the target profile in a new tab
  • ↻: Manual ping now (immediate activity check)
  • ⚔: Open attack page in a new tab
  • x: Remove target from tracked list

Manual ping has anti-spam cooldown protection to prevent rapid repeated requests.

Notifications and Toasts

Minerva can notify on tracked status changes.

  • Visible tab: in-page Minerva toast (draggable, closable)
  • Hidden tab: browser notification if available, otherwise Tampermonkey notification
  • Cooldown spacing to reduce alert spam
API Key and Passphrase

Minerva needs a Torn API key (recommended access: User -> Profile).

  • Custom Minerva API key entry popup
  • Optional encrypted key vault (Web Crypto)
  • Passphrase unlock prompt every 7 days
  • Forgot-passphrase reset flow (clears stored vault and lets you re-enter a key)
Troubleshooting and Compatibility
  • Built-in Minerva log panel for request and UI diagnostics
  • Works across Torn pages with a floating widget
  • Uses multiple profile-panel injection strategies (known selectors, anchors, dynamic matching, generic fallback)
  • Filters extension-origin errors out of Minerva logs to reduce noise
